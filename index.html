      Uso de LaTeX

      Motor de Retención: FSRS-6

      "Estudiador Pro" no usa fechas fijas. Utiliza el Free Spaced Repetition Scheduler (FSRS), un modelo matemático que calcula el decaimiento exacto de tus sinapsis en base a 21 parámetros de memoria. En particular este programa utiliza parámetros medios obtenidos por grupos de investigación a lo largo de millones de repasos y miles de usuarios. A la larga, se pueden recopilar los datos para producir parámetros personales y conseguir resultados óptimos.

      El propósito de este modelo es predecir el momento exacto en el que deberías volver a repasar un concepto en base a su dificultad, tu historial de recuperación de contenidos, tu estabilidad, entre muchos otros.

      Las 3 Fases del Aprendizaje:
      • Nueva (New): Primer contacto. El algoritmo aún no conoce tu retención para este concepto.
      • Reaprendizaje (Learning): Tarjetas falladas. Intervalos muy cortos (minutos/horas) para forzar la reconexión neuronal.
      • Repaso (Review): Memoria consolidada. Intervalos largos (días/meses) diseñados para interrumpir la curva del olvido justo antes de que el concepto desaparezca.
      Cómo calificar (Botones):
      1. Fácil Respuesta trivial e instantánea. Multiplica drásticamente el próximo intervalo.
      2. Bien (Botón por defecto). Recordada con fluidez normal. Mantiene el ritmo óptimo de retención.
      3. Difícil La recordaste, pero con un esfuerzo mental severo o duda extrema.
      4. Crítica Amnesia total. Fuerza la tarjeta de vuelta a la fase de Reaprendizaje.
      Widget de Distribución de Tarjetas

      El widget muestra en qué fase del aprendizaje se encuentra cada tarjeta según el algoritmo FSRS-6:

      Nuevas — Nunca repasadas. El algoritmo no tiene datos sobre ellas.
      Reaprendizaje — Falladas recientemente. El algoritmo las reprograma con intervalos cortos para reforzar la conexión.
      Consolidando — En repaso espaciado pero con estabilidad baja (menos de 7 días). Aún frágiles.
      Revisión — Estabilidad media (7-21 días). Bien asentadas pero con revisión periódica necesaria.
      Dominadas — Estabilidad alta (más de 21 días). El concepto está sólidamente en memoria a largo plazo.
